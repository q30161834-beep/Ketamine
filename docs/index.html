Ketamine

A compiled, statically-typed systems language focused on performance and security.

⚡ LLVM-native · 🛡 Memory-safe · 🎯 Multi-backend
GitHub Getting Started Docs Playground

De ce Ketamine?

⚡ Viteză

Compilat la LLVM IR, apoi la cod mașină nativ. Zero runtime overhead, optimizări LLVM.

🛡 Siguranță

Ownership + borrowing (ca Rust), fără garbage collector, fără null, fără buffer overflows.

🎯 8 Backend-uri

LLVM, Go, Python, JavaScript, WASM, Native x86-64, ASM, Forth — același cod, orice platformă.

📦 Ecosistem

Package manager, LSP server, VS Code extension, playground in-browser, debugger DWARF.

🔧 Tooling

Formatter, documentation generator, REPL cu JIT, profiler, code coverage, test framework.

🌐 FFI

Zero-copy FFI cu C, Python (numpy), Go, Rust. Ownership policies: copy, pin, transfer, refcount.

Exemplu rapid

// Funcție generică cu const generics
fn sum_array<T, const N: int>(arr: [T; N]) -> T where T: Add {
    let mut sum = 0
    for i in 0..N {
        sum = sum + arr[i]
    }
    return sum
}

// Async function
async fn fetch_data(url: str) -> str {
    let resp = await http::get(url)
    return resp.body
}

fn main() -> int {
    let future = fetch_data("https://api.example.com")
    let result = Runtime::block_on(future)
    print(result)
    return 0
}

Instalare rapidă

# One-liner
curl -fsSL https://raw.githubusercontent.com/q30161834-beep/Ketamine/main/scripts/install.sh | bash

# Sau manual
git clone https://github.com/q30161834-beep/Ketamine.git
cd Ketamine
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)

# Rulează primul program
./ketc ../examples/00_hello.kt --target jit
