# Tutorials — Step by Step Guide

## Tutorial 00: Hello, World!

**File:** `examples/00_hello.kt`

Run:
```bash
ketc examples/00_hello.kt --target go -o hello.go
go run hello.go
```

## Tutorial 01: Variables

**File:** `examples/01_variables.kt`

Covers: `let`, `mut`, `const`, type annotations, shadowing.

## Tutorial 02: Functions

**File:** `examples/02_functions.kt`

Covers: `fn`, parameters, return, implicit return, recursion.

## Tutorial 03: Control Flow

**File:** `examples/03_if_while.kt`

Covers: `if/else`, `while`, `for`, `loop`, `break`, `continue`, `match`.

## Tutorial 04: Structs

**File:** `examples/04_structs.kt`

Covers: `struct`, fields, methods (`self`, `&mut self`), static methods.

## Tutorial 05: Enums

**File:** `examples/05_enums.kt`

Covers: `enum`, variants with data, `match` with patterns.

## Tutorial 06: Generics

**File:** `examples/06_generics.kt`

Covers: generic functions, generic structs, where clauses.

## Tutorial 07: Traits

**File:** `examples/07_traits.kt`

Covers: `trait`, `impl Trait for Type`, trait bounds, default methods.

## Tutorial 08: Standard Library

**File:** `examples/08_stdlib.kt`

Covers: `import`, Vec, Option, Result.

## Tutorial 09: HTTP Server

**File:** `examples/09_http_server.kt`

Covers: structs, Vec, routing, request/response handling.

## Tutorial 10: Tic-Tac-Toe

**File:** `examples/10_game.kt`

Covers: full game with enums, match, loops, Vec, structs, methods.
